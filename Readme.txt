This project focuses on performing exploratory data analysis on a given dataset using the R programming language. The objective is to derive insights from the data and identify any patterns or trends that can be used to inform decision-making.

The project is structured as follows:

1.Importing the necessary libraries and reading the dataset
2.Handling missing values
3.Descriptive statistics of the dataset
4.Aggregating, filtering, and ranking the data
5.Data visualization
6.Insights from the data
7.Installation

The project requires R programming language and the following packages: dplyr, and ggplot2. These can be installed using the following commands:

install.packages("dplyr")
install.packages("ggplot2")


Usage

To run the project, open the R console and type the following command:

source("file_path/exploratory_analysis.R")
Replace file_path with the path to the R file containing the code.
